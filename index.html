Bike Lanes vs Car Lanes in Sweden

Washington Post recently published a great set of maps showing only the bike lanes for Washington DC, Boston, Miami and Seattle.

Inspired by this, Andrew J. Timmons decided to use Google Maps to show bike lanes and car lanes for the top 20 most populated cities in America. Kristofer Palmvik forked the code to create similar maps of the ten largest cities in Sweden.
